# 社区项目

项目介绍：本项目是一个基于Java的前后端不分离的社区项目。支持帖子发布、查看、编辑、评论、点赞；用户关注、发送私信等功能。

技术栈：Spring Boot、Spring、Spring MVC、MyBatis、Redis、Kafka、Thymeleaf

功能介绍：
- 社区登录模块：
  - 发送邮件
  - 注册功能
  - 会话管理
  - 生成验证码
  - 登录|退出功能 
  - 显示登录信息
  - 账号设置
  - 检查登陆状态

- 社区核心模块：
  - 过滤敏感词
  - 发布帖子
  - 帖子详情
  - 事务管理
  - 显示评论
  - 添加评论
  - 私信列表
  - 发送私信
  - 统一处理异常
  - 统一记录日志
  - 点赞
  - 查询收到的赞
  - 关注|取消关注
  - 关注列表|粉丝列表
  - 发送系统通知
  - 显示系统通知
